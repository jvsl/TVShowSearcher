# TVShowSearcher

## Architecture
- MVVM

## Frameworks
- Combine
- SwiftUI
- Kingfisher

## Tests

The viewModel TVShowListTests owns tests to give examples of how test the others viewModels.

## Installation

Just open TVShowSearcher.xcodeproj using XCode and press run.
